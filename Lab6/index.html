¿Cuáles son las diferencias entre los posibles valores de la propiedad position?

Ele valor relative posiciona un elemento respecto al flujo normal de la página. Absolute, toma como referencia la ventana del navegador o del elemento posicionado. Fixed, toma como referencia la ventana del navegador y no respetan el tener un contenedor padre que esté posicionado. Inherit, hereda el valor para esta propiedad del elemento padre. Initial, toma su valor por defecto. Sticky, toma el valor hasta que se alcanza un umbral de desplazamiento.

¿Cuáles son los valores estándar para la propiedad visibility?

Se usa para dos efectos, el valor hidden esconde un elemento, pero deja vacío el espacio donde debería de estar. Y el valor collapse esconde filas o columnas de una tabla.

¿Qué es el zIndex y para qué sirve?

Indica el z-order de un elemento y sus descendientes. Cuando varios elementos se superponen, z-order determina cuales cubren a otros. Un elemento con mayor z-index generalmente cubre a otro menor.
