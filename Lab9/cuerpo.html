  • ¿Qué hace la función phpinfo()?

    Muestra la informacion sobre el sistema operativo que se esta utilizando, sobre el servidor apache, la version con la que actualmente se esta trabjando como la información sobre PHP y enseña los valores de la configuracion que son por default y en el valor que actualmente estan.

  • ¿Qué cambios tendrías que hacer en la configuración del servidor para que pudiera ser apto en un ambiente de producción?

    Copiar el paquete web en el servidor, se descrgar xampp para que la computadora actue como servidor.

  • ¿Cómo es que si el código está en un archivo con código html que se despliega del lado del cliente, se ejecuta del lado del servidor? Explica.

    El servidor se encarga de ejecutarlo y cuando lo envia al cliente este se encarga de interpretarlo. Y lo que hace PHP al final es transformar y mandarlo con formato HTML para ser interpretado por el cliente.
