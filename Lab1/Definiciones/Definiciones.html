Tecnología Definición
URL Dirección única que poseen todos los elementos en internet, la cual permite ubicarlos fácilmente.
MIME Es un standard que sirve para identificar los tipos de datos que son enviados a través de la red.
HTTP Protocolo que garantiza una comunicación efectiva entre servidores, clientes y proxies.
HTML5 Lenguaje estandar para diseñar y presentar contenido web.
JavaScript Lenguaje utilizado para crear acciones y que posee menos restricciones.
PHP Código abierto utilizado principlamente para crear páginas web dinámicas.
Java Server Pages Tecnología para crear aplicaciones web compatibles con múltiples plataformas.
AJAX Unión de diferentes programas creada para mejorar la interacción del usuario.
