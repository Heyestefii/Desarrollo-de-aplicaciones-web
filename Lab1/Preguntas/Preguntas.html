¿Cuál es la diferencia entre Internet y la World Wide Web?

La World Wide Web permite la navegación a través de páginas web, mientras que el internet proporciona los recursos necesarios.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Get se utiliza para adquirir un archivo o recurso de un servidor web. Head obtinene sólo los metadatos del archivo o recurso. Post sirve para enviar información o elemento, la cual posteriormente se guardara como un subelemento. Put se encarga de crear o cargar un nuevo recurso. Patch modifica parcialmente un recurso. Delete solicita borrar recursos especificos.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Post, por la seguridad y tamaño de la información.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

hola

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

No, significa que la petición fue recibida correctamente.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Si, porque significa que existe algún enlace roto, y debe reparase.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Si, ya que un error 500 se genera por errores dentro de la programación de la plataforma.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

Significa que ya no forma parte del nuevo estándar, debido a que es remplazado o se vuelve obsoleto. Ejemplos: applet, dir, insindex, basefont, etc.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

Se agregan o cambian nuevos atributos, elementos, como: media, section, type, etc. Se mejoran las APIS getElementsByClassName() y innerHTML.

¿Qué componentes de estructura y estilo tiene una tabla?

Se utilizan las etiqueta table, tr y th, para indicar que es una tabla, la separación de columnas y filas, así como el contenido de esta.

¿Cuáles son los principales controles de una forma HTML5?

Párrafo, artículo, tablas, listas, etc.
